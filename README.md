## :rocket: Getting Started

1. Download the repository

- Using Git

```shell
  git clone https://github.com/gledson-ss/netflix-frontend

```

1. Installation

- Using `yarn`

```shell
yarn
```


1. Execution Server

  cd server
  yarn start
